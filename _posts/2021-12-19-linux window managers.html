JWM - Joe's window manager:


This has always been one of my favourite window managers. It's very easy to use, runs really fast. And does everything one could need. The only thing I would say is that it doesn't come with a autogenerated menu,out of the box, neither does it arrange windows in a automatic way.

Jwm is a manual tiling window manager, which means you decide where the windows go and how they display.

I have created a .jwmrc config file, which you need to place in your ./home folder, and the icons needed can be added by placing the jwm folder in 'usr/share/jwm'

in order to get that autogenerated menu, all you need to do is follow my .readme file and this will help you to install xdgmenmaker which will then give you a dynamic menu that will automatically add apps to the menu, without the need to manually configure the menu.
